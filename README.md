#  DevClub — Exercício 3

Página desenvolvida como exercício do curso **DevClub**, utilizando HTML e CSS para praticar estruturação, estilização e organização de assets.

![Ilustração do projeto](./assets/MulherCelular.svg)

## Tecnologias utilizadas

- HTML5
- CSS3

## Estrutura do projeto

```text
.
├── index.html
├── styles.css
└── assets/
    └── MulherCelular.svg
```

## Como executar

1. Baixe ou clone este repositório.
2. Abra o arquivo `index.html` no seu navegador.

Não é necessário instalar dependências.

## Objetivo do exercício

Praticar a construção de uma interface web com HTML e CSS, incluindo o uso de imagens SVG e a organização dos arquivos do projeto.

Projeto realizado durante o curso [DevClub](https://devclub.com.br/).
